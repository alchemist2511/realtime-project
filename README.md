# Finhub Data Pipeline

This project is a **data pipeline** built on **Finnhub.io Trading APIs & WebSocket**  
for collecting, processing, and visualizing real-time stock data.

## Environments

You need to set the following environment variables:
create file ".env" and set your FINNHUB_TOKEN.

```bash
FINNHUB_TOKEN=your_finnhub_token
```
